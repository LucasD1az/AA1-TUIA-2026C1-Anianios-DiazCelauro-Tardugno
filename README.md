## Configuración del Entorno

Para ejecutar este proyecto, se recomienda crear un entorno virtual para mantener las dependencias aisladas.

### En Windows:
1. Abrir la terminal (PowerShell o CMD) en la carpeta del proyecto.
2. Crear el entorno: `python -m venv venv`
3. Activar el entorno: `.\venv\Scripts\activate`
4. Instalar librerías: `pip install -r requirements.txt`

### En Linux / Ubuntu:
1. Abrir la terminal en la carpeta del proyecto.
2. Crear el entorno: `python3 -m venv venv`
3. Activar el entorno: `source venv/bin/activate`
4. Instalar librerías: `pip install -r requirements.txt`
